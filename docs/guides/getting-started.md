## What is Refract?

Refract is an open-source, OpenTelemetry-native observability platform for AI systems. It helps you:

- **Track costs** - Monitor LLM API costs in real-time
- **Monitor quality** - Detect semantic degradation and hallucinations
- **Test changes** - Replay production traffic before deploying
- **Stay compliant** - Self-host with full data control

## Self-Hosted Features

The free self-hosted tier includes:

- **50,000 traces per day** - Resets daily at midnight UTC
- **7-day retention** - Automatic cleanup of old traces
- **All features** - Alerts, replay testing, semantic scoring, and more
- **Zero lock-in** - OpenTelemetry-compatible, works with your existing stack
